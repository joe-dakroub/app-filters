# Filters

An image filter application built with Vite and Vanilla JavaScript.

## Features

- ASCII art filter
- Dithering effects
- Duotone filter
- Image enhancement
- Linocut effect
- Sketch filter

## Installation

```bash
# Install dependencies
yarn install
```

## Development

```bash
# Start development server
yarn dev
```

## Build

```bash
# Build for production
yarn build
```

## Preview

```bash
# Preview production build
yarn preview
```
